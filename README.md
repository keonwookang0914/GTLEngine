# Pikachu Volleyball (DirectX)

DirectX 기반으로 제작한 피카츄 배구 카피 게임입니다.  
2인 플레이와 AI 상대 플레이를 지원합니다.

## 게임 조작 방법

### 둘이서 하기

**1P 조작키**

- `W A S D` : 이동 / 점프  
- `Space` : 스파이크

**2P 조작키**

- `방향키` : 이동 / 점프  
- `Enter` : 스파이크

### 혼자서 하기

**플레이어**

- `방향키` : 이동 / 점프  
- `Enter` : 스파이크

**AI**

- 2P 캐릭터를 AI가 자동으로 조작합니다.

---

## 플레이 방법

플레이어는 피카츄를 조작하여 공을 상대 코트로 넘겨 득점하는 게임입니다.  
공이 상대 코트 바닥에 닿으면 점수를 획득합니다.

점프와 스파이크를 활용하여 공의 방향과 속도를 조절할 수 있습니다.

## 주요 feature

### 물리 상호작용
- 공과 캐릭터 간 **Circle Collision** 처리입니다.  
- 지형 및 네트 충돌을 위한 **AABB Collision** 처리입니다.  
- 충돌에 따른 공 반사 및 물리 반응 구현입니다.

### 피카츄 캐릭터 조작
- 방향키 : 좌우 이동 기능입니다.
- 위쪽 방향키 : 점프 기능입니다.
- 점프 후 Space : 스파이크 공격 기능입니다.

### 피카츄 AI
- 공 위치 기반 이동 로직입니다.  
- 점프 타이밍 판단 로직입니다.  
- 공을 넘기기 위한 공격 동작 로직입니다.

### 애니메이션
- 캐릭터 상태 기반 스프라이트 애니메이션입니다.  
- 이동 / 점프 / 공격 애니메이션 전환입니다.

### 키보드 입력 처리
- 실시간 키 입력 처리입니다.
- 2인 플레이 입력 분리입니다.
